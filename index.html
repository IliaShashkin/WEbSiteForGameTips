GameTips

Project Overview: Welcome to GameTips, a site dedicated to providing an introduction to popular games, along with tips and recommendations to help players improve. Whether you're a beginner or an experienced player, this site offers insights on gameplay mechanics and strategies for popular games like War Thunder, Valorant, and The Finals.

Pick a Game

The Finals Valorant War Thunder

Objectives

  • Provide a basic summary of each game listed: War Thunder, Valorant, and The Finals.
  • Offer useful tips for each game to help players improve.
  • Give recommendations on how to start as a beginner in each game.

Features

  • Homepage: A sidebar with links to detailed pages about each game, encouraging users to explore and learn more.
  • Navigation Menu: A sidebar on the left that allows users to navigate between game pages.
  • Additional Pages/Sections: Each game will have its own dedicated webpage with more in-depth content, including tips and strategies.

Technologies Used

  • HTML for structure and content.
  • CSS for styling and layout.

Deliverables

  • All source code files (HTML, CSS).
  • Documentation detailing the structure of the website and design choices.
